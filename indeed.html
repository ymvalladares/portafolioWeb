Book Store App
BookStore is a comprehensive C# and React desktop application designed to provide bookstore owners with a powerful tool for managing inventory, processing orders, and enhancing customer satisfaction through efficient and user-friendly functionalities.

Key features:

  • Security: Auntentication and Authorization with Json Token Bearer
  • Inventory Management: BookStore allows owners to maintain a digital catalog of books, including details such as title, author, genre, ISBN, price, quantity in stock, and supplier information. It supports adding, editing, and deleting books from the inventory, as well as tracking stock levels and generating inventory reports.
  • Order Processing: BookStore enables bookstore staff to process customer orders efficiently, from order placement to fulfillment and delivery. It includes features for capturing order details, managing order status (e.g., pending, processing, shipped), generating invoices, and updating inventory levels upon order fulfillment

Tecnologies:

  • C#, Entity Framework, MySql, Stripe, React
